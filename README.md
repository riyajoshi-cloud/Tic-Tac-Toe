# Tic-Tac-Toe
This project is a simple Tic Tac Toe game implemented in Python that allows two players to play the game in the console.
<br>
The program creates a 3×3 grid where players take turns placing their symbols (X and O) on the board. The game checks for winning conditions after each move, such as completing a row, column, or diagonal. If all spaces are filled without a winner, the game ends in a draw.
<br>
The project demonstrates basic Python concepts like loops, conditional statements, functions, and list operations while implementing simple game logic. It is useful for beginners to understand how game flow and user interaction can be handled in Python.
